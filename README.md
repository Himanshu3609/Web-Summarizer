# Web Summarizer – Chrome Extension

Web Summarizer is a Chrome extension that automatically extracts article content from any webpage and generates a concise summary using AI.

## 🚀 Features

- Extracts main article text from web pages
- Generates short, readable summaries
- Clean and simple popup interface
- Fast and lightweight
- Works on most blog and news websites

## 🛠 Tech Stack

- JavaScript
- Chrome Extension APIs
- HTML & CSS
- Gemini API (for AI summarization)

## ⚙️ How It Works

1. The extension uses a content script to extract article text.
2. The text is sent to the AI API.
3. The AI returns a summarized version.
4. The summary is displayed inside the popup UI.

## 📦 Installation (Developer Mode)

1. Clone this repository
2. Open `chrome://extensions`
3. Enable Developer Mode
4. Click "Load Unpacked"
5. Select this project folder

## 🎯 Use Case

Useful for:
- Students
- Researchers
- Developers
- Anyone who wants quick insights from long articles
